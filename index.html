nym_assignment

A PDF Parser for Nym's Home Assignment

View on GitHub

Overview

This repository is a solution for parsing PDF documents using pdfplumber. The main tasks include:

  • pdf_to_dict: Converts a PDF into a dictionary with page numbers as keys and lists of TextualWord objects as values.
  • populate_chart: Extracts patient data (name, DOB, EKG validity) and computes the patient's age.
  • pdf_to_extra_dict: Extracts additional details (font name and size) into ExtraTextualWord objects.
  • split_to_sections: Splits extra words into sections based on bold titles.

Installation

Ensure you have Python 3.8+ installed, then run:

pip install pdfplumber

To test the functionality, run:

python assignment_api.py

Usage

All main functions and objects are in assignment_api.py. The repository includes several PDF files:

  • chart1.pdf
  • chart2.pdf
  • chart3.pdf
  • chart_example.pdf

Run assignment_api.py to see parsing results and section splits.

Code Explanation

The code is divided into several tasks:

  1. PDF to Dictionary: Extracts words from the PDF into a dictionary of TextualWord objects.
  2. Populate Chart: Extracts patient details to create a Chart object with an age calculation property.
  3. Extra Textual Words: Extracts font information into ExtraTextualWord objects.
  4. Split to Sections: Groups extra words into sections using bold tokens as headers.
