nym_assignment

Advanced PDF Parser for Automated Medical Coding, built on the Nym Health approach.

Open GitHub

Overview

This repository uses pdfplumber to parse PDF documents for the Nym home assignment. It automates extracting patient details and includes the following tasks:

  • pdf_to_dict: Converts PDF pages to TextualWord objects.
  • populate_chart: Extracts name, DOB, EKG status, and computes the patientâ€™s age.
  • pdf_to_extra_dict: Captures font info using ExtraTextualWord.
  • split_to_sections: Groups data into sections based on bold titles.

Installation

Requires Python 3.8+ and pdfplumber:

pip install pdfplumber

Then run:

python assignment_api.py

Usage

All core functions and classes are in assignment_api.py. PDF files:

  • chart1.pdf
  • chart2.pdf
  • chart3.pdf
  • chart_example.pdf

The program prints parsing results and demonstrates chart population and section splitting.

Code Explanation

Core components:

  1. pdf_to_dict: Basic word extraction.
  2. populate_chart: Builds a Chart (name, DOB, EKG, age).
  3. pdf_to_extra_dict: Enriched extraction into ExtraTextualWord.
  4. split_to_sections: Groups bold-based headings into logical sections.
